# TODOLIST
To-Do List Web Application built using HTML, CSS, and JavaScript as a JavaScript mini project. This application allows users to add tasks with deadlines, mark tasks as completed, highlight overdue tasks, and delete tasks when no longer needed. The project demonstrates core concepts of DOM manipulation, event handling, and input validation.

# To-Do List App

A simple To-Do List web application built with HTML, CSS, and JavaScript.

## Features
- Add tasks with deadline
- Highlight overdue tasks
- Mark tasks as completed
- Delete tasks
- Enter key support

## Tech Stack
- HTML
- CSS
- JavaScript
